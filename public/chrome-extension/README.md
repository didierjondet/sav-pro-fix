# Extension Chrome - Recherche Pièces Fournisseurs

Cette extension Chrome permet de rechercher des pièces chez Mobilax et Utopya directement depuis votre navigateur.

## Installation

1. Ouvrez Chrome et allez à `chrome://extensions/`
2. Activez le **Mode développeur** (bouton en haut à droite)
3. Cliquez sur **Charger l'extension non empaquetée**
4. Sélectionnez le dossier `chrome-extension` (là où se trouve ce README)
5. L'extension est maintenant installée !

## Utilisation

1. Cliquez sur l'icône de l'extension dans la barre d'outils Chrome
2. Sélectionnez les fournisseurs (Mobilax et/ou Utopya)
3. Entrez votre recherche (ex: "écran iPhone 12")
4. Cliquez sur **Rechercher**

## Fonctionnalités

- Recherche sur Mobilax et Utopya en parallèle
- Affichage des prix et disponibilités
- Copie des données produit en JSON
- Lien direct vers la fiche produit

## Conseils

- Connectez-vous à vos comptes fournisseurs dans Chrome pour voir les prix professionnels
- La recherche utilise votre session navigateur, donc les prix affichés sont ceux de votre compte

## Création des icônes

Pour que l'extension fonctionne correctement, créez des fichiers icônes :
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

Vous pouvez utiliser n'importe quelle image de votre choix ou créer un simple carré coloré avec un outil comme Paint.
